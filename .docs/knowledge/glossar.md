# Glossar

- **Mandant/Tenant**: isolierter Daten- und Berechtigungsraum einer Kanzlei oder Einzelorganisation.
- **TemplateVersion**: unveränderliche Version eines Musterdokuments.
- **ClauseVersion**: unveränderliche Version einer Klausel.
- **Pinning**: Vertragsobjekt referenziert exakt eine Template-/Clause-Version.
- **Guided Contract Builder**: geführter Q&A-Prozess zur Vertragserstellung.
- **Release Gate**: definierter Freigabeschritt vor Veröffentlichung von Inhalten.
