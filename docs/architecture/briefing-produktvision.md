# Briefing Snapshot – Produktvision

Quelle: `docs/architecture/legaltech-produkt-briefing.md`

## Vision
Mandantenfähige Legal-Tech-Plattform (Cloud + On-Prem), in der Verlage Muster pflegen und Kanzleien daraus guided Verträge erzeugen.

## Rollen
- Plattform-Admin
- Verlag-Admin
- Autor/Redakteur
- Mandant/Kanzlei
- Kanzlei-Org-Admin (optional)

## Kernfähigkeiten
- Musterkatalog + Versionierung
- Guided Q&A mit Konsistenzsicherung
- Dokumentexport (Word/LibreOffice)
- Spätere API-/Signaturintegration
