# Team 02 – Platform Security & Identity

## Auftrag
Sichert Mandantenfähigkeit, Rollenmodell, Zugriffskontrolle, Auditierbarkeit und DSGVO-Konformität.

## Scope
- Tenant Isolation (serverseitig erzwungen)
- RBAC (Admin/Editor/Nutzer)
- Audit-Events, Session-Policies, MFA-Option
- Datenschutz- und Löschkonzepte

## Deliverables
- Security Baseline v1
- IAM/RBAC-Modell
- Audit- und Compliance-Checkliste
